# Void-2: The Linguistic Model

> 100MB. Eloquent. Creative. Human.

## Overview
Void-2 is the linguistic engine of Void AI. Built on Phi-3-mini and fine-tuned for 
creative and analytical writing, it writes like a human but thinks like a system.

## Capabilities
- **Creative Writing**: Stories, essays, poetry, scripts, articles
- **Analysis**: Multi-perspective breakdowns, critical thinking, nuanced arguments
- **Conversation**: Natural dialogue with context retention
- **Void Code Literacy**: Can read/explain basic Void Code

## Quick Start (Ollama)
```bash
ollama create void-2 -f Modelfile
ollama run void-2 "Write a short story about a programmer who discovers Void Code"
```

## Resource Requirements
| Resource | Minimum | Recommended |
|----------|---------|-------------|
| RAM | 2 GB | 4 GB |
